## 🚀 Featured Projects

### 🎥 [Real-Time Sign Language Recognition Platform](https://github.com/kr-aashay)
An enterprise-grade video conferencing architecture designed to bridge communication gaps for deaf users in live streams.
* **Core Architecture:** Engineered a dual-recognition pipeline featuring browser-side MediaPipe processing (WebAssembly at 30fps) paired with a server-side PyTorch Bi-LSTM + CTC model.
* **Performance:** Achieved an optimized **1–3 second gesture-to-caption latency** with 91.2% accuracy on the ISL-CSLTR dataset.
* **Integration:** Utilized FastAPI and WebSockets for real-time bidirectional streaming, Gemini 2.0 Flash for NLP-driven sentence generation from raw sign glosses, and MongoDB for persistent session history.

### 🎙️ [Interactive Voice Agent for Event Management](https://github.com/kr-aashay)
A zero-interface, zero-form telephonic automation system handling end-to-end event discovery and management.
* **Core Architecture:** Engineered a high-throughput asynchronous WebSocket bridge using Python `asyncio` to connect Twilio streams with Deepgram STT/TTS and GPT-4o-mini.
* **Features:** Implemented dynamic real-time intent recognition, voice-driven CRUD operations via LLM function calling, and an intelligent caller-ID caching layer for returning users.

### 📊 [Retail Sales Forecasting Pipeline](https://github.com/kr-aashay)
An end-to-end machine learning pipeline built to process and forecast massive commercial datasets.
* **Model Engine:** Trained a LightGBM model on 913K rows of data using strict leakage-free, year-lagged rolling features, achieving a highly optimized validation RMSE of 8.20.
* **Analytics Layer:** Deployed an interactive Streamlit dashboard visualizing store-by-item sales heatmaps, seasonal day-of-week trends, and detailed forecast curves for complete interpretability.
